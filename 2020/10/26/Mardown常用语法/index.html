标题

“#” 字符的数量对应标题的等级，1-6，记得后面加个空格

段落

markdown 使用空白行分割段落

越努力，越幸运

越自律，越自由

换行

要创建换行符号,请以两个或者多个空格结束一行

着重显示

可以使用粗体或者斜体来着重显示你想要的突出的文字

粗体

可以在词句两端添加 ** 或者 __ 来实现,如果是下划线，两边需要有空格
我是要成为海贼王的男人
我是要成为 海贼王 的男人
我要着重强调这一行

斜体

同粗体一样，只不过是换成了一个下划线或者星号，此时下划线两边可以不加空格
比萨斜塔
比萨斜塔

又粗又斜

同时实现粗体和斜体，使用三个下划线
这句话真的真的很重要

块引用

要创建块即引用，在段落前面加 __ > __

孔子说，人是铁饭是钢；
老子说，孔子说的对，一顿不吃饿得慌

块引用嵌套

通过右尖括号的数量来表示嵌套的层级

下面一句话正反念都一样

上海自来水来自海上

块引用和其他表示结合起来使用

有序列表

很简单，记得句子前面需要有空格

  1. 打开冰箱门
  2. 把大象放进冰箱
  3. 关上冰箱门

可以有缩进

  1. 学习笔记
    1. python
    2. javascript
    3. nodejs
  2. 生活感悟
    1. 如何提高自己的收入水平
    2. 努努力
  3. 点滴记录

无序列表

有三种方式，在句子前面添加 * 、+ 、-

  • 第1步
  • 第2步
  • 第3步

代码行

代码块通常需要缩进一个制表符四个空格,但是需要新起一个段落，也就是在上部插入一个空行

<div> 你好，世界 <div>
const word = 'Hello world'
